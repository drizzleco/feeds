A Flask app with Test Suite(tox, pytest), Docker, Heroku, Flask-Login, Vue.js(CDN Version), jQuery, Sass, and SQLite(Flask-SQLAlchemy)

# Getting Started

## `make install`

- install dependencies

## `make start`

- start flask server

## `make lint`

- lint code with black

## `make test`

- run tox and pytest to test code
